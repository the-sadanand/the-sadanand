# images/

Place your profile images here. Recommended files:

| Filename              | Purpose                        | Suggested Size     |
|-----------------------|--------------------------------|--------------------|
| `profile-banner.png`  | Hero banner for preview.html   | 900 × 300 px       |
| `profile-photo.png`   | Your avatar / headshot         | 400 × 400 px       |
| `hero-background.png` | Dark nebula background         | 1920 × 1080 px     |
| `architecture.png`    | System architecture diagram    | Any                |
| `timeline.png`        | Career timeline graphic        | Any                |
| `ml-pipeline.png`     | ML pipeline diagram            | Any                |
| `favicon.png`         | Browser tab icon               | 32 × 32 px         |

> All images in this folder are referenced from `preview.html`.
> The GitHub `README.md` uses external image URLs (shields.io, capsule-render) instead.
