# gifs/

Place animated GIFs here. Recommended files:

| Filename          | Purpose                             | Max Size |
|-------------------|-------------------------------------|----------|
| `coding.gif`      | Coding / typing animation           | 2 MB     |
| `ai-animation.gif`| Neural network / AI visualisation   | 2 MB     |
| `typing.gif`      | Keyboard / terminal typing effect   | 1 MB     |
| `stars.gif`       | Star / nebula background loop       | 2 MB     |

> Tip: Use [ezgif.com](https://ezgif.com) to compress GIFs before adding them.
> GitHub has a 10 MB file limit per file but keep GIFs under 2 MB for fast loading.
